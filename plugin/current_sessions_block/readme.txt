<h1 class="page-header">Current Sessions Block</h1>

<p>This plugin adds a block with the current sessions for a user.</p>
<p>For each session block show the progress and stars achieved for each courses in session.</p>

<h2>Configuration</h2>
<p>You must set a region to this plugin</p>

<ul>
    <li>
        <strong>Show block</strong>
        Show the session block when is setted to true.
    </li>
    <li>
        <strong>Number of sessions</strong>
        Number of session to show in the block.
    </li>
    <li>
        <strong>Days before to start a session</strong>
        Number of days to consider before start of sessions.
    </li>
</ul>
