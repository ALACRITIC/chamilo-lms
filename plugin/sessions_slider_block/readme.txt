<h2 class="page-header">Sessions Slider Block Plugin</h2>
<p>Add a slider with course sessions on the site's homepage.</p>

<p>Require set the <code>homepage_bottom</code> region to the plugin.</p>

<p>This plugin add the show_in_slider and video_url_in_slider extra fields to determine which sesssion be displayed in the slider.</p>
